# Entregas de analise de algoritmos

Esta pasta contem os artefatos necessarios para a entrega de perfilamento e analise de algoritmos do MVP Hand Rehab.

## Estrutura

- `relatorio/final.md`: relatorio final de perfilamento e analise.
- `relatorio/Otimização_de_Telemetria_com_Buffer_Circular.pdf`: versao em PDF do relatorio.
- `evidencias/`: CSVs com as execucoes e resultados reais dos benchmarks.
- `evidencias/README.md`: descricao dos dados presentes nos CSVs.
- `graficos/real/`: graficos usados pelo relatorio final.
- `perfilamento/build_real_report_assets.py`: script para regenerar os graficos reais a partir dos dados da execucao de referencia.

## Observacoes

- A execucao de referencia do relatorio e `762ba7a6-41a5-42ee-99fa-b5a1c23023f6`.
- As evidencias principais desta entrega estao em `evidencias/benchmark_runs.csv` e `evidencias/benchmark_results.csv`.
- Os graficos devem permanecer sincronizados com os dados descritos em `relatorio/final.md`.
- Nao incluir rascunhos, evidencias intermediarias ou artefatos de entregas anteriores nesta pasta.
