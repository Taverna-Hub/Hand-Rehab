# Perfilamento

Esta pasta contem apenas o script necessario para regenerar os graficos do relatorio final.

## Script

- `build_real_report_assets.py`: gera os graficos em `../graficos/real/` a partir dos dados da execucao real `762ba7a6-41a5-42ee-99fa-b5a1c23023f6`.

## Uso

```bash
MPLCONFIGDIR=/tmp/matplotlib-cache .venv/bin/python entregas/perfilamento/build_real_report_assets.py
```
