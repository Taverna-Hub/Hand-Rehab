# Evidencias

Use esta pasta para registrar comprovacoes de execucao do sistema.

Exemplos:

- Prints do Node-RED recebendo mensagens MQTT.
- Logs do Serial Monitor da ESP32.
- Saidas de `docker compose up --build`.
- Respostas de endpoints do backend.
- Evidencias de dados persistidos no Postgres.
- Prints do frontend base aberto em `http://localhost:5173`.
